## Your Schools

### Description

Your Schools is a MERN stack application that allows users to find preschools and daycares in their area. It provides comprehensive information about each school, including location, hours, and tuition.

### Technologies Used

- MongoDB
- TypeScript
- Express
- React
- Node.js
- GraphQL
- Google Maps API
- Tailwind CSS
- DaisyUI
- JWT
